Indicates the original media type of the resource (prior to any content encoding applied for sending).

## Description
In responses, it tells the client what the content is (e.g. `application/json`, `text/html`). In POST/PUT requests, it tells the server what type of data is being submitted.

## Usage Note
Headers are case-insensitive, although they are conventionally written in `Train-Case`.
