Informs the server about the types of data that can be sent back in the response.

## Description
Example: `Accept: text/html, application/xhtml+xml, application/xml;q=0.9, image/webp, */*;q=0.8`

## Usage Note
Headers are case-insensitive, although they are conventionally written in `Train-Case`.
